## Overview
Cross-platform micro-investment app built with Flutter. Start investing with just $1. Features AI financial coaching, auto round-up savings, and budget tracking.

## Tech Stack
- Flutter 3.x, Riverpod, Firebase
- FL Chart for financial visualizations
- Green/Blue gradient design for trust

## Features
- Round-up savings automation
- Micro-investing (start with $1)
- AI financial coach
- Budget tracker with insights
- Goal visualization

## Monetization
- Freemium (basic free)
- Premium: $6.99/month
- Commission: 0.25% on returns
